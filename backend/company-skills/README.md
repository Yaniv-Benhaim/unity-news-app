# Company AI Skills

This backend project includes a small internal skill pack to demonstrate how a company could standardize AI-assisted development across Android teams.

In production these skills would be published from a shared company registry. They are included here so reviewers can see the intended operating model.

## Skills Included

- `backend-service-change`: change the backend service or foreground-service behavior safely.
- `backend-filter-contract`: add filters while keeping the UI/backend contract stable.
- `android-security-review`: review exported components, IPC validation, and release hardening.
